# 🧠 Quiz App

## 📌 Description
A responsive Quiz Web Application built using HTML, CSS, and JavaScript.  
It allows users to answer multiple-choice questions and receive an instant score at the end.

## 🚀 Features
- Multiple-choice questions  
- Real-time score calculation  
- Result summary screen  
- Responsive UI  

## 🛠️ Technologies Used
- HTML  
- CSS  
- JavaScript
